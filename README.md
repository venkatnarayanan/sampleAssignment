# sampleAssignment
### Pre-requisite:
- ```nvm install 10.24.0```
- ```npm install -g ember-cli@2.9.1```
- ```npm install -g bower```
### Inside the project, run this command:
- ```npm install```
- ```ember s```
### If there is any errors:
- Delete node_modules/
- Delete package.lock.json
- then try ```ember s```
